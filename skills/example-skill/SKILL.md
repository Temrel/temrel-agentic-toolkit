# Example skill

This is a template. Copy this folder, rename it, and edit the frontmatter and body below to create a real skill.

## When to use this

Describe the trigger: the kind of task or request that should make an agent load this skill. Be concrete. An agent decides whether to pull in a skill based mostly on the `description` line in the frontmatter and this section, so make both unambiguous.

## Instructions

Write the actual know-how here, as direct instructions to the agent. Keep it focused on one capability. A good skill body:

1. States the goal in a sentence.
2. Lists the steps to follow, in order.
3. Calls out the common mistakes to avoid.
4. Shows a short example of a correct result.

Keep it short. A skill is meant to be pulled in on demand, so it should add exactly the knowledge needed for its task and nothing more. If you find yourself writing a lot of background, that background probably belongs in a doc the skill links to, not in the skill itself.

## Example

Show one small, concrete example of the skill applied correctly, so the agent has a pattern to match.
