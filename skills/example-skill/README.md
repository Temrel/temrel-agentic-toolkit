# example-skill

**What:** A stub agent skill and `SKILL.md` template you copy to create new skills.

**Why:** So a new skill can be dropped into `skills/` in minutes, with a consistent shape that agents can discover and load.

## How to use it

1. Copy this `example-skill/` folder.
2. Rename it to your skill's name.
3. Edit `SKILL.md`: set the `name` and `description` in the frontmatter, then write the body.
4. Update this README with your skill's real what, why, and issue link.

See the [skills folder convention](../README.md) for the full layout.

## Introduced in

This template ships with the initial scaffold of the toolkit. Future skills should link here to the Temrel issue that introduced them.
