# Guardrails and security

One of the six elements of agentic engineering. This is about **keeping a capable agent safe to run**: sandboxing, approval gates, permission scoping, secret handling, and checks that catch dangerous or unintended actions before they happen. The more autonomy you grant an agent, the more this element earns its keep.

## What belongs here

Sandboxing setups, approval-gate patterns, permission and allowlist configs, secret-scanning hooks, prompt-injection defences, and pre-flight safety checks for agent actions.
